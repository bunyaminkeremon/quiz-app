# Quiz App

A timed quiz application that fetches questions from the Open Trivia Database API. Pick a category and difficulty, answer 10 questions within 30 seconds each, and see your final score.

## Live Demo

[Try it here](https://bunyaminkeremon.github.io/quiz-app/)

## What It Does

- 8 different categories (General Knowledge, Computers, Sports, History, Science, Geography, Film, Music)
- 3 difficulty levels (Easy, Medium, Hard)
- 30-second countdown timer per question
- Live score tracking
- Color-coded feedback on answers
- Final score with performance rating

## Tech Stack

- HTML5
- CSS3
- JavaScript (async/await, Fetch API)
- Open Trivia Database API (no key required)

## Run Locally
```bash
git clone https://github.com/bunyaminkeremon/quiz-app.git
```

Open `index.html` in your browser.
